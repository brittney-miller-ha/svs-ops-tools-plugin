# SVS Ops Tools

Claude skills for the SVS team at Instacart.

## Install

Open Cowork and paste this message to Claude:

> Install the SVS ticket filing skill from https://raw.githubusercontent.com/brittney-miller-ha/svs-ops-tools-plugin/main/file-svs-ticket.skill

Claude will fetch the skill and show an install button. Click **Save skill** and restart Cowork.
